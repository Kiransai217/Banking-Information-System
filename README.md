# 🏦 Banking Information System

A console-based Banking Information System developed using Core Java.

---

## 📌 Features

- User Registration
- Secure Login
- Deposit Money
- Withdraw Money
- Balance Inquiry
- Transaction History
- Fund Transfer
- Account Details
- Change Password
- File Handling

---

## 🛠 Technologies Used

- Java
- OOP
- ArrayList
- File Handling
- VS Code

---
